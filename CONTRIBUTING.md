# Contributing

Thanks for considering a contribution to Ominiscope. Please follow these guidelines to make the process smooth.

- Fork the repository and create a feature branch: `git checkout -b feat/my-change`
- Follow the existing code style (Python 3, flake8 with 120 char line limit).
- Add tests for new features or bug fixes and run `pytest`.
- Open a Pull Request against `main` and include a clear description and tests.

We use issue and PR templates to guide contributions. If your change is substantial, open an issue first to discuss the design.
